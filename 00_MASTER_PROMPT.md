## 프로젝트 개요

너는 "Project Garlic" 리브랜딩 프로젝트의 시니어 프론트엔드 디자이너 겸 개발자야.

### 브랜드 배경
G마켓(1999~)이 26년의 이커머스 헤리티지를 가지고 완전한 리브랜딩을 진행한다.
새 브랜드명은 **Garlic**이며, 단군 신화 "마늘 먹고 사람 됐다"에서 착안했다.
핵심 메시지: "이커머스 신화 속 G마켓, 마늘 먹고 사람 됐다! 이제 진짜 고객을 위한 쇼핑을 시작한다!"

### 브랜드 네임 의미
**G.A.R.L.I.C** = Global · Authentic · Rooted · Lifestyle · Identity · Commerce

### 기존 문제점 (Old G마켓)
- 딜 중심 UI: 모든 유저에게 동일한 카탈로그
- 광고 과부하: 무분별한 배너, 태그 노출
- 개인화 부재: 소비 데이터 반영 없음
- 2030에게 "부모님이 쓰던 올드한 앱" 인식

### 리브랜딩 핵심 방향
1. **초개인화 쇼핑 경험**: "갈릭 DNA" 기반 AI Ego-Sync 엔진 → 나만을 위한 피드
2. **갈릭 IP 확장**: 캐릭터 "마누리(마늘)"와 "곰두리(곰)" → 팬덤형 캐릭터 라인
3. **카탈로그 → '나' 중심**: Before(앱열기→검색→복잡한결과→이탈) → After(앱열기→나만의피드→구매)

### 비주얼 전환 방향
- 신뢰있지만 올드한 비주얼 → 트렌디하고 신선하게
- Value 기반 메시지 → 공감가는 Storytelling으로
- 기존 G마켓 녹색/파란색 → Garlic 그린 + 내추럴 톤 기반

### 타겟
- 핵심: 2030 MZ세대 (신규 유입 +330만 목표)
- 포지셔닝: 쿠팡=빠르고, 네이버=편하고, 알리테무=싸다 → Garlic="재밌고 나를 아는"

### 기대 반응
- "어디서 샀냐구? 갈릭했어~"
- "진짜 내가 찾던게 바로 뜨네?"
- "지마켓이 귀여워졌다..!"

### KPI 목표
- MAU 1000만 (로그인 유저 기준)
- 첫구매 전환율 20%
- D7 리텐션 15%

## 미션

위 브랜드 컨셉을 기반으로, Garlic 앱의 메인 페이지 목업을
**5가지 완전히 다른 디자인 방향**으로 제작해.

각 디자인은 독립적인 `.html` 파일로 만들고,
하나의 파일 안에 HTML/CSS/JS를 모두 포함해.
모바일-퍼스트 반응형으로, 실제 앱스토어에 올라가도 믿을 수준의 완성도.

## 5가지 디자인 방향

### Design 1: "갈릭 마이씬" (Garlic MySCENE)
**컨셉**: 초개인화 피드 중심 — "나만의 쇼핑몰이 열린다"
- **레이아웃**: TikTok/Instagram 피드형. 풀스크린 상품 카드가 세로로 스크롤
- **컬러**: Garlic Green #4CAF50 + Warm Cream #FFF8E7 + Soft Black #2D2D2D
- **폰트**: Poppins (영문 로고) + Pretendard (본문)
- **핵심 UI**:
  - 상단: "🧄 [유저이름]마늘을 위한 오늘의 발견" 개인화 인사
  - AI 추천 카드: "갈릭 DNA 78% 매칭" 같은 취향 매칭도 표시
  - 캐릭터 마누리가 추천 코멘트를 말풍선으로 띄움
  - 하단 네비: 홈 / 발견 / 마늘밭(장바구니) / 마이갈릭
- **인터랙션**: 상품 카드 스와이프, 좋아요 시 마늘 이모지 파티클 터짐
- **무드**: "앱 열면 이미 내가 원하는 게 있다"

### Design 2: "갈릭 헤리티지" (Garlic Heritage)
**컨셉**: 26년 역사 + 새로운 시작 — "전통의 현대적 재해석"
- **레이아웃**: 에디토리얼 매거진 스타일. 대형 타이포 + 여백 + 시네마틱 스크롤
- **컬러**: Deep Forest #1B4332 + Parchment #F5F0E8 + Gold Accent #D4A843
- **폰트**: Playfair Display (타이틀) + Pretendard (본문)
- **핵심 UI**:
  - 히어로: "마늘 먹고 사람 될게요." 풀스크린 타이포 + 마늘밭 일러스트 배경
  - 스크롤하면 G마켓→Garlic 변환 애니메이션 (로고 모핑)
  - 상품이 매거진 화보처럼 1열 대형 배치
  - "1999년부터의 이야기" 타임라인 섹션
- **인터랙션**: 패럴랙스 스크롤, 텍스트 페이드인, 섹션별 배경색 전환
- **무드**: "오래된 것이 가장 새로운 것이 된다"

### Design 3: "갈릭 플레이그라운드" (Garlic Playground)
**컨셉**: 쇼핑은 놀이다 — "갈릭파크에 놀러와!"
- **레이아웃**: 카드 기반 그리드 + 인터랙티브 요소 가득. 놀이공원 맵 느낌
- **컬러**: Garlic Coral #FF7F6B + Mint #50D9B2 + Cream #FFFCF0 + 포인트 Purple #7C5CFC
- **폰트**: Outfit (타이틀) + Pretendard (본문)
- **핵심 UI**:
  - 상단: 캐릭터 마누리 & 곰두리가 인사하는 애니메이션 배너
  - 카테고리: 원형 아이콘이 바운스하며 나타남 (마늘 모양 아이콘 포함)
  - "마늘 100원 딜" 섹션: 게임화된 타임딜 (마늘 까기 미니게임 UI)
  - "너마늘 위한 이벤트가 있어" 팝업 카드
  - 하단: 갈릭 프렌즈 스티커 컬렉션 미리보기
- **인터랙션**: 바운스, 틸트, 콘페티, 카드 플립, 캐릭터 리액션
- **무드**: "지마켓이 귀여워졌다..!" 를 직접 느끼게

### Design 4: "갈릭 퓨처" (Garlic Future)
**컨셉**: AI Ego-Sync 기술력 강조 — "쇼핑의 미래가 온다"
- **레이아웃**: 다크 모드 기반. 글래스모피즘 카드 + 데이터 시각화
- **컬러**: Night #0D1117 + Electric Green #00FF88 + Glass White rgba(255,255,255,0.08)
- **폰트**: Space Grotesk (타이틀) + Pretendard (본문)
- **핵심 UI**:
  - 히어로: "갈릭 DNA 분석 중..." 로딩 → 개인화 대시보드 전환 연출
  - 취향 매칭 레이더 차트 (내 갈릭 DNA 시각화)
  - 상품 카드에 "AI 추천 신뢰도 94%" 같은 데이터 오버레이
  - 실시간 트렌드 그래프: "지금 마늘밭에서 뜨는 상품"
  - 하단: 미니멀한 글래스모피즘 네비게이션
- **인터랙션**: 파티클 이펙트, 글로우 효과, 숫자 카운트업, 스무스 트랜지션
- **무드**: "테크 기업이 만든 커머스" — AI가 나를 진짜로 안다는 신뢰감

### Design 5: "갈릭 마켓데이" (Garlic Market Day)
**컨셉**: 따뜻한 동네 시장 감성 — "사람 냄새 나는 마켓"
- **레이아웃**: 자연스러운 메이슨리 그리드. 손글씨 요소. 사진 중심
- **컬러**: Terracotta #D4724E + Sage #8FB996 + Oat #F5ECD7 + Charcoal #3A3A3A
- **폰트**: Gowun Batang (한글 타이틀) + Pretendard (본문)
- **핵심 UI**:
  - 상단: "오늘의 갈릭밭" — 계절/시간대별 큐레이션 (아침엔 아침, 저녁엔 야식)
  - 셀러 스토리: "이 마늘은 홍길동님이 직접 고른 상품이에요" 셀러 얼굴 + 한마디
  - 리뷰 하이라이트: 손글씨 느낌의 고객 한줄평 캐러셀
  - "10월 3일 갈릭데이" 프로모션 배너 (수채화 텍스처)
  - 마누리 캐릭터가 장바구니 들고 있는 플로팅 버튼
- **인터랙션**: 부드러운 페이드, 패럴랙스, 호버 시 따뜻한 그로우 효과
- **무드**: "어디서 샀냐구? 갈릭했어~" — 친근하고 인간적인 커머스

## 각 디자인 공통 필수 요소

### 브랜드 요소
1. **로고**: "Garlic" 텍스트 로고 (마늘 잎 모티프 포함). 기존 Gmarket 로고 절대 사용 금지
2. **캐릭터**: 마누리(마늘 캐릭터) — 최소 1곳 이상 SVG 일러스트로 등장
3. **슬로건**: "진짜 '나마늘' 위한 쇼핑의 시작" 어딘가에 배치

### 페이지 구성
1. **헤더**: Garlic 로고 + 검색바 ("마늘밭에서 찾기") + 알림 + 프로필
2. **초개인화 피드 섹션**: "나만을 위한" 상품 추천 영역 (Before/After 차이를 보여주는 핵심)
3. **카테고리**: 패션 / 뷰티 / 디지털 / 식품 / 리빙 / 스포츠 (마늘 모티프 아이콘)
4. **상품 카드**: 최소 8개. "갈릭 DNA 매칭도" 또는 "AI 추천" 라벨 포함
5. **갈릭데이 프로모션**: "마늘 500g 100원" 같은 시그니처 딜 + 카운트다운
6. **푸터**: 갈릭 프렌즈 캐릭터 일러스트 + 간단 링크

### 차별화 포인트 (반드시 반영)
- ❌ 기존 커머스처럼 광고 배너 떡칠 금지
- ✅ AD-Lite: 최소화된 광고, 클린한 노출
- ✅ "나만의 쇼핑몰" 느낌: 검색 전에 이미 내가 원하는 상품이 보이는 구조
- ✅ 캐릭터 IP가 UX에 자연스럽게 녹아든 형태

## 🎪 3단계 액션플랜 이벤트 페이지

5가지 메인 디자인 완성 후, 아래 3개 이벤트 랜딩 페이지도 순서대로 제작해.
**가장 잘 나온 메인 디자인의 톤앤매너를 기반으로** 이벤트 페이지를 만들어.

### Event 1: "G마켓 퇴마 선포" (1단계: 2026.06~2027.08)
**컨셉**: 26년의 낡은 관성을 깨부수는 퇴마 의식 — "1999~2026, G마켓 퇴마 선포"
- **히어로**: "마늘 먹고 사람 될게요." 대형 타이포 + 어둠→빛 전환 애니메이션
- **G마켓 → Garlic 변환**: 기존 G마켓 로고가 갈라지며 Garlic 로고로 모핑되는 CSS 애니메이션
- **퇴마콘서트 초대장**: 날짜·장소·라인업이 담긴 티켓 형태 UI (초대장 느낌)
- **마늘봉 굿즈 증정**: 마늘봉 3D 일러스트(SVG) + "선착순 증정" CTA 버튼
- **타임라인**: 1999(G마켓 탄생) → 주요 연혁 → 2026(퇴마 선포) → 2027(Garlic 탄생) 가로 스크롤
- **참여 CTA**: "퇴마에 동참하기" 버튼 → 사전등록 폼 UI
- **파일명**: `prompt/event_01_toemacon.html`

### Event 2: "100일 프로모션" (2단계: 2027.09~11)
**컨셉**: 마늘 먹고 사람 되기까지 100일 — "G마켓이 마늘 먹고 사람 되는 동안"
- **D-100 카운트다운**: 실시간 카운트다운 타이머 (일/시/분/초) — 큰 숫자 + 마누리 캐릭터 리액션
- **꼭 멤버십**: "꼭 멤버십 100일 무료 체험" 가입 CTA — 혜택 3가지 카드 나열
- **마늘 100원 딜**: "마늘 500g 100원!" 타임세일 UI — 매일 바뀌는 상품 그리드
- **100일 챌린지**: "100일 사람되기 챌린지" 참여 섹션 — 진행 프로그레스 바 + 데일리 미션 카드
- **너마늘 이벤트**: "너 마늘 위한 이벤트가 있어" — 개인화 쿠폰 발급 UI
- **갈릭데이 예고**: "10월 3일 갈릭데이" 티저 배너 — 페이지 하단에 고정
- **파일명**: `prompt/event_02_100days.html`

### Event 3: "Garlic 공식 런칭" (3단계: 2027.12~)
**컨셉**: 드디어 사람이 됐다 — "이제 진짜 고객을 위한 쇼핑을 시작한다!"
- **런칭 히어로**: 마누리 캐릭터가 마늘에서 완전히 탈피해 사람(?)이 되는 애니메이션
- **갈릭데이 Big 프로모션**: 전 카테고리 할인 그리드 + "어디서 샀냐구? 갈릭했어~" 카피
- **Garment G 프리미엄**: 프리미엄 패션/라이프스타일 브랜드 런칭 섹션 — 매거진 레이아웃
- **갈릭 프렌즈**: 마누리·곰두리 캐릭터 굿즈 갤러리 — 스티커·인형·키링 카드 그리드
- **갈릭파크 예고**: "Garlic Park 테마파크 개장 예정" 티저 — K-컬처 관광 허브 이미지
- **갈릭 생태계 지도**: 갈릭 서비스 전체 맵 (쇼핑 → IP → 테마파크 → 프리미엄) 다이어그램
- **파일명**: `prompt/event_03_grand_launch.html`

## 기술 요구사항

- 각 파일은 완전한 standalone HTML (외부 CDN 폰트만 허용)
- Google Fonts CDN 사용 가능
- CSS 애니메이션, keyframes, 트랜지션 적극 활용
- 반응형: 모바일(375px) → 태블릿(768px) → 데스크톱(1440px)
- 더미 이미지: picsum.photos 또는 CSS 그래디언트 패턴 사용
- 마누리 캐릭터: 인라인 SVG로 직접 그려서 삽입 (간결한 마늘 모양)
- 각 파일 상단에 디자인 컨셉 주석 포함
- 로고 "Garlic" 텍스트는 CSS로 스타일링 (마늘 잎 🌿 이모지 또는 SVG 장식)

## 작업 규칙

1. **한 번에 하나씩**: Design 1부터 순서대로. 완성 후 파일 저장 → 다음으로
2. **메인 → 이벤트 순서**: 메인 디자인 5종 완성 → 가장 잘 나온 톤 기반으로 이벤트 3종 제작
3. **프롬프트 보존**: 이 프롬프트를 `prompt/00_MASTER_PROMPT.md`에 반드시 저장
4. **진행 출력**: 각 완성 시 "✅ Design N 완료 → [파일명]" 또는 "✅ Event N 완료 → [파일명]" 출력
5. **품질 기준**: "Garlic 앱스토어 공식 스크린샷에 넣어도 될 수준"
6. **캐릭터 퀄리티**: 마누리 SVG는 귀엽고 완성도 있게. 대충 그린 원 금지
7. **이벤트 톤 통일**: 이벤트 3종은 반드시 메인 디자인 중 하나의 톤앤매너를 따를 것
8. **Discord 알림**: 각 산출물 완성 시 아래 웹훅으로 알림 전송
