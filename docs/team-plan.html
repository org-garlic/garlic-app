01프로젝트 개요

Ego-Sync — 유저가 앱을 여는 순간, 앱이 유저가 된다

갈릭마켓의 핵심 차별화 전략은 유저별 완전 개인화 UI입니다. 동일한 앱이라도 각 유저의 DNA 프로필에 따라 레이아웃·컴포넌트 배치·CTA 문구·색조까지 달라집니다. Ego-Sync는 이 초개인화를 구현하는 풀스택 AI 시스템입니다.

03갈릭 DNA 엔진

6축 사용자 프로파일링

유저의 구매·탐색·리뷰 데이터를 6개 축으로 압축한 DNA 벡터를 생성합니다. 각 축의 가중치는 카테고리 전략에 따라 조정되며, 매일 배치 + 세션 내 실시간 보정이 결합됩니다.

축 (Axis) 카테고리 주요 데이터소스 가중치
AXIS 1 패션 / 뷰티 구매 이력, 위시리스트, 리뷰 감성 0.25
AXIS 2 디지털 / 가전 검색 쿼리, 비교 클릭, 구매 전환 0.20
AXIS 3 식품 / 신선 정기구독, 반복구매, 배송지 패턴 0.20
AXIS 4 리빙 / 인테리어 PIN 저장, 장바구니 이탈, 시즌 구매 0.15
AXIS 5 스포츠 / 아웃도어 브랜드 선호, 사이즈 필터, 재구매율 0.12
AXIS 6 취미 / 기타 롱테일 클릭, 이벤트 참여, 소셜 공유 0.08
배치 학습
플랫폼: Alibaba PAI
스케줄: 매일 새벽 3AM
학습 데이터: 2026년 전체 로그
모델: Collaborative Filtering + Embedding
실시간 보정
스트리밍: Alibaba Flink
캐시: Redis Cluster
갱신 주기: 세션 내 이벤트 발생 시
반영 항목: 클릭·카트·검색
벡터 DB
엔진: AnalyticDB for PG
유사도: cosine similarity
임베딩 상품 수: 100만 개
차원: 768d
